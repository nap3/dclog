# dclog################################
SQL Serverのデータベースの変更履歴を取得できるようにするツール

## Description
データが変更されたテーブルの一覧と変更前と後のデータを記録する。

## Features
* データが変更されたテーブルの一覧
* テーブル毎の変更前と変更後のデータ一覧

##Requirement
* SQL Server
